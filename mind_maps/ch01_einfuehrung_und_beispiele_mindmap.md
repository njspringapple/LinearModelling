# 第 1 章：导论与例子

## 回归的基本对象
- 目标变量 $Y$
  - 课程中要解释、检验或预测的变量
- 解释变量 $x$
  - 用来解释 $Y$ 的变量
  - 可扩展为多个解释变量 $x_1,\dots,x_k$
- 随机误差 $\varepsilon$
  - 表示模型没有解释掉的随机波动
  - 说明 $Y$ 和 $x$ 的关系通常不是确定性的

## 回归的基本形式
- 一般形式
  - $Y=f(x)+\varepsilon$
  - $E(Y\mid x)=f(x)$
- 简单线性起点
  - $E(Y\mid x)=\beta_0+\beta_1x$
  - $\beta_0$ 表示截距
  - $\beta_1$ 表示平均趋势的斜率
- Galton 例子
  - $Y$: 孩子身高
  - $x$: 父母平均身高
  - 关键现象: regression toward the mean
  - 直觉: 极端父母的孩子平均更靠近总体均值

## 回归的三个目标
- 理解 / Verstehen
  - 哪些变量影响 $Y$
  - 影响方向是什么
  - 影响大小如何解释
- 检验 / Nachweisen
  - 某个影响是否有统计证据
  - 典型问题: $\beta_j$ 是否为 $0$
  - 关注假设检验和置信区间
- 预测 / Prognostizieren
  - 给定新个体的 $x_0$ 预测 $Y_0$
  - 点预测: $\widehat{Y}_0$
  - 需要考虑预测不确定性

## 讲义例子对应的模型族
- Galton 身高
  - 连续 $Y$
  - 连续 $x$
  - 简单线性回归
- Midlife crisis
  - 连续响应变量
  - 多个解释变量
  - 年龄效应可能非线性
  - 多元线性回归 + splines
- Umweltzone
  - 连续污染物浓度
  - 政策阶段是分组变量
  - 温度等变量是协变量
  - 协方差分析 / ANCOVA
- 教育阅读能力
  - 学生嵌套在班级中
  - 班级之间可能有随机差异
  - 混合线性模型 / mixed model
- 慢性支气管炎 CBR
  - 二元响应变量 $Y\in\{0,1\}$
  - 目标是患病概率
  - logistic 回归
  - $P(Y=1)=G(\eta)$

## 选择模型的判断顺序
- 先看 $Y$ 的类型
  - 连续变量
  - 二元变量
  - 计数变量
  - 分位数问题
- 再看解释变量结构
  - 连续解释变量
  - 分类解释变量
  - 连续变量与分类变量混合
- 再看数据结构
  - 是否有分组
  - 是否有嵌套
  - 是否有重复测量
  - 是否可能相关
- 最后看关系形状
  - 近似线性
  - 非线性
  - 需要 polynomial
  - 需要 spline

## 建模前的探索性数据分析
- 单变量分布
  - 看 $Y$ 和每个 $x$ 的范围
  - 看异常值
  - 看偏态和分布形状
- 连续变量关系
  - 散点图
  - 趋势形状
  - 异常点
  - 是否可能非线性
- 分类变量关系
  - 分组箱线图
  - 组间差异
  - 组内波动
- EDA 对模型选择的提示
  - 非线性提示变换、polynomial 或 spline
  - 方差不恒定提示 WLS / GLS
  - 分组明显提示 ANOVA / ANCOVA / mixed model

## 记号约定
- 随机变量
  - $Y_i$
  - 用于讨论期望、方差和概率性质
- 观测值
  - $y_i$
  - 已经在数据中看到的值
- 解释变量
  - $x_i$
  - 本章先看作给定的观测协变量
- 学习时的区分
  - 推导中保留 $Y_i$
  - 数据分析中常用 $y_i$
  - 实际分析中使用数据列名
